# PoE2 Builds

Pesquisador de builds de Path of Exile 2 — busca as melhores builds nas principais fontes da comunidade.

## Como usar
1. Abra `index.html` no navegador
2. Selecione uma classe (Bruxa, Feiticeira, Patrulheira, etc.)
3. Clique em **Pesquisar**
4. Veja as melhores builds ranqueadas com links diretos

## Fontes
Maxroll, Mobalytics, PoE Ninja, Reddit (r/PathOfExile2Builds)

## Classes
8 disponíveis + 4 bloqueadas (em breve)

## Tecnologia
HTML puro + JavaScript (single-file, sem dependências)
